bluetooth

Bluetooth is a short-range wireless technology that connects devices like smartphones, laptops, headphones, keyboards, and speakers. It allows data transfer without cables using radio waves. Bluetooth is commonly used for audio streaming, file sharing, and device pairing. It consumes low power and works within approximately 10 meters range

wifi

Wi-Fi is a wireless networking technology that connects devices like smartphones, laptops, smart TVs, and routers to the internet without cables. It uses radio waves to transmit data over short to medium distances. Wi-Fi provides high-speed internet access and supports multiple devices simultaneously within homes, offices, and public places.

webpages

Webpages are digital pages on the internet that display information using text, images, videos, and links. They are created using languages like HTML, CSS, and JavaScript. Webpages are accessed through web browsers such as Google Chrome, Mozilla Firefox, and Microsoft Edge. Multiple webpages together form a website.

websites

Websites are collections of related webpages stored on a web server and accessible through the internet. They provide information, services, or entertainment. Websites are built using technologies like HTML, CSS, and JavaScript. Examples include Wikipedia, Amazon, and YouTube. Users access websites using web browsers on computers or mobile devices.
